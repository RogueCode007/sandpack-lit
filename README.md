# Sandpack Lit

A lit implementation of "Sandpack React".

The aim is to build a reduced-scoped api of the official sandpack-react repo.

## Components
### `<sandpack-preset>`

options:
- template: string
- files
  - code: string
- customSetup
  - dependencies: string
  - externalResources: string[]
- initMode: "lazy" | "immediate"
- closableTabs: boolean
